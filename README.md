# Unemployment Analysis in India

This project analyzes unemployment trends across different regions in India using Python and data visualization techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning
- Handling missing values
- Exploratory Data Analysis
- Unemployment trend visualization
- Region-wise unemployment comparison

## Key Insight
Tripura, Haryana, and Jharkhand show the highest unemployment rates among Indian regions.
